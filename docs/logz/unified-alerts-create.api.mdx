# Create Unified Alert

Create a new unified alert.

### Endpoint
`POST /event-manager/v2/unified-alerts/{type}`

### Description
This endpoint allows you to create a new unified alert.

### Parameters
- **type** (path): The type of the unified alert (e.g., `LOG_ALERT`, `METRIC_ALERT`).

### Request Body
The request body should include the unified alert configuration.

### Response
Returns the details of the created unified alert.
